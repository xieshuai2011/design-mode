抽象工厂模式

抽象工厂模式是一个超级工厂，用来创建其他工厂。 这个工厂也被称为工厂的工厂。
在抽象工厂模式中，接口负责创建相关对象的工厂，而不明确指定它们的类。 每个生成的工厂可以按照工厂模式提供对象。

 提供一个创建一系列相关或相互依赖的对象的接口 ， 无需指定它们具体的类：

```java
public interface AbstractInternetFactory {
    /**
    * 获取品牌
    * @return
    */
    Brand getBrand();
    /**
    * 获取中国阿里巴巴
    * @return
    */
    Alibaba getAlibaba();
    /**
    * 获取外国亚马逊 
    * @return  
    */
    Amazon getAmazon();
}
```

实现接口工厂的具体工厂：
```java
public class ChinaFactory implements AbstractInternetFactory{
    @Override    public Brand getBrand() {
        return new DongFeng();//具体产品
    }
    @Override
    public Alibaba getAlibaba() {
        return new Alibaba();//具体产品
    }
    @Override
    public Amazon getAmazon() {
        return null;//具体产品
    }
}
public class ForeignFactory implements AbstractInternetFactory{
    @Override
    public Brand getBrand() {
        return new Nike();//具体产品
    }

    @Override
    public Alibaba getAlibaba() {
        return null;//具体产品
    }

    @Override
    public Amazon getAmazon() {
        return new Amazon();//具体产品
    }
}
```

测试结果：

```java
public class AbstractFactoryTest {
    public static void main(String[] args) {
        AbstractInternetFactory chinaFactory = new ChinaFactory();
        Brand brand = chinaFactory.getBrand();
        Alibaba alibaba = chinaFactory.getAlibaba();
        Amazon amazon = chinaFactory.getAmazon();
        System.out.println("中国工厂有如下产品：");
        print(brand);
        print(alibaba);
        print(amazon);
        System.out.println("---------------------");
        AbstractInternetFactory foreignFactory = new ForeignFactory();
        brand = foreignFactory.getBrand();
        alibaba = foreignFactory.getAlibaba();
        amazon = foreignFactory.getAmazon();
        System.out.println("外国工厂有如下产品：");
        print(brand);
        print(alibaba);
        print(amazon);
    }
    static void print(Brand brand) {
        if(brand == null){
            System.out.println("无此产品");
        }else {
            System.out.println(brand.getName());
        }
    }
}

中国工厂有如下产品：
东风
阿里巴巴互联网
无此产品
---------------------
外国工厂有如下产品：
耐克
无此产品
亚马逊互联网
```

当抽象工厂模式中的产品等级只有一层的时候，工厂方法也就只有一个，这时抽象工厂模式就等同于工厂方法模式。
当工厂模式中的所有产品实例都交给了一个具体工厂统一创建时，抽象工厂类不再需要并且将工厂方法定义为静态方法，这时工厂模式就等同于简单工厂模式。
